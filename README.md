# Color-Recall
A browser-based color sequence memory game created as part of my JavaScript learning journey. Inspired by classic pattern-memory games, this project was built purely for educational and practice purposes to improve frontend development and logical thinking skills.

An interactive memory game where players must remember and repeat increasingly difficult color sequences. Built as part of my JavaScript learning journey to practice game logic, event handling, and dynamic UI interactions.

The project recreates a classic pattern-memory style game with responsive feedback and increasing difficulty levels.
The game challenges players to remember and repeat increasingly complex color sequences while providing interactive visual feedback.

Note:

This project is not an original game concept and was recreated for learning and practice purposes while exploring JavaScript and interactive web development.


Features:
Randomized color sequence generation,
Increasing difficulty with each round,
Interactive button animations,
Score and progress tracking,
Responsive UI design,
Real-time user input validation
